## CLAUDE_DESKTOP_REDESIGN.md
Полный редизайн ais152.com. Запускается единым промптом, внутри 3 фазы (P0 аудит → P1 дизайн → P2 билд) с гейтами `=== WAITING_FOR_CEO_OK ===` между ними.

**Использование:**
1. Перед вставкой подставь `{TODO_WHATSAPP}` (твой номер без `+`) и `{TODO_EMAIL_FORMSUBMIT}` (email для формы)
2. Открой Claude Desktop → новый чат
3. Скопируй весь файл → вставь
4. Жди roadmap → дай OK → жди дизайн-варианты → выбери A/B/C → жди билд

Если контекст оборвётся — напиши «продолжи с фазы P1» (или P2).
